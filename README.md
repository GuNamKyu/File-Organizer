# 📂 File-Organizer

사용자가 제공한 경로의 파일을 **확장자 기반 분류 규칙**에 따라 자동으로 정리하는 AI Agent 워크플로우 프로젝트입니다.

## 주요 기능

- 📁 **확장자 기반 자동 분류**: 파일 확장자에 따라 카테고리(문서, 이미지, 영상 등)로 분류
- 🔢 **폴더 넘버링**: 폴더명에 `01_`, `02_` 형식의 번호 자동 부여
- 📄 **파일명 유지**: 파일 이름은 변경하지 않고 원래 이름 그대로 이동
- 🏗️ **폴더 자동 생성**: 필요한 분류 폴더가 없으면 규칙에 맞게 자동 생성
- 🔒 **범위 제한**: 지정된 경로 내부만 수정 (외부 접근 없음)

## 사용법

AI Agent에게 다음 형식으로 경로를 제공합니다:

```
[경로 : C:\Users\example\Downloads]
```

Agent가 파일을 스캔하고 정리 계획을 보여준 뒤, 승인 시 실행합니다.

## 분류 규칙

| 카테고리 | 확장자 예시 |
|----------|------------|
| 문서 | `.pdf`, `.doc`, `.docx`, `.txt`, `.hwp`, `.xlsx`, `.pptx` |
| 이미지 | `.jpg`, `.png`, `.gif`, `.bmp`, `.svg`, `.webp` |
| 영상 | `.mp4`, `.avi`, `.mkv`, `.mov`, `.wmv` |
| 음악 | `.mp3`, `.wav`, `.flac`, `.aac`, `.ogg` |
| 압축 | `.zip`, `.rar`, `.7z`, `.tar`, `.gz` |
| 코드 | `.js`, `.ts`, `.py`, `.java`, `.html`, `.css` |
| 실행파일 | `.exe`, `.msi`, `.dmg`, `.apk` |
| 데이터 | `.sql`, `.db`, `.sqlite` |
| 폰트 | `.ttf`, `.otf`, `.woff`, `.woff2` |
| 기타 | 매핑되지 않은 확장자 |

## 핵심 규칙

1. **폴더만 넘버링** — 파일명은 절대 변경하지 않음
2. **최대 5단계** 폴더 깊이 제한
3. 분류 폴더가 없으면 **자동 생성**
4. 지정 경로 **내부만** 수정
